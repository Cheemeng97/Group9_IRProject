# Group9_IRProject
 python crawler

# Setup
```
conda create -n EncyclopediaCrawler python=3.6 
conda activate EncyclopediaCrawler
conda install -c conda-forge EncyclopediaCrawler 
```

# Start env
```
conda activate EncyclopediaCrawler
```

# Stop env
```
conda deactivate
```
# Run spider
Go to the project top level directory and run
```
scrapy crawl "name"
```
